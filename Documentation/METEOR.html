First run

File system or server

To run Meteor version you will need Meteor package installed. Please go to Meteor website to install package for your system: https://www.meteor.com/install
Before you run Luna please be sure that you can create and run sample Meteor app: https://www.meteor.com/try Please be sure that you don't have any error while running sample Meteor app.

If everything work well please go to Luna_Full_Version_Meteor (Luna Meteor folder) and run meteor command 

meteor

Next open your web browser and go to. 

http://localhost:3000

And after that you will see the app running. If you have any problem on this stage feel free to write to us on support@webapplayers and we will try to help.

Layout structure

Luna layout

Main layout are created with few main elements:

  1. .wrapper main page wrapper
  2. {{> header }} top header with menu
  3. {{> navigation }} left sidebar navigation.
  4. .content {{>Template.dynamic template=content}} main container for page elements.
<template name="layout">

    <!-- Wrapper-->
    <div class="wrapper">

        <!-- Header-->
        {{> header }}

        <!-- Navigation-->
        {{> navigation }}

        <!-- Main content-->
        <section class="content">

            <!-- Main view  -->
            {{>Template.dynamic template=content}}

        </section>
        <!-- End main content -->

    </div>
    <!-- End wrapper-->

</template>

Page header

Custom page header

Each page has also own pageHeader template - it is a heading bar with breadcrumb It is added to view as a {{> pageHeader}}

{{> pageHeader category='UI Elements' section='General' title='Alerts' icon="pe-7s-speaker" desc='Notification and custom alerts' }}

And the structure of it is:

    <div class="row">
        <div class="col-lg-12">
            <div class="view-header">
                <div class="pull-right text-right" style="line-height: 14px">
                    <small>{{category}}<br>{{section}}<br> <span class="c-white">{{title}}</span></small>
                </div>
                <div class="header-icon">
                    <i class="pe page-header-icon {{icon}}"></i>
               </div>
                <div class="header-title">
                    <h3>{{title}}</h3>
                    <small>
                        {{desc}}
                    </small>
                </div>
            </div>
            <hr>
        </div>
    </div>

Route config

Flow router

To manage all route we use great plugin Flow router https://github.com/kadirahq/flow-router. It allows you to organize the parts of your interface into a templates. Additional to have a layout feature we use BlazeLayout: https://github.com/kadirahq/blaze-layout. Below you can see example of configuration routes. Configuration routing are in client->config->router.js


FlowRouter.route('/', {
    action: function() {
        FlowRouter.go('/dashboard');
    }
});

FlowRouter.route('/dashboard', {
    action: function() {
        BlazeLayout.render("layout", {content: "dashboard"});
    }
});


FlowRouter.route('/metrics', {
    name: 'metrics',
    action: function() {
        BlazeLayout.render("layout", {content: "metrics"});
    }
});

// UI Elements
// General

FlowRouter.route('/panels', {
    name: 'Panels',
    action: function() {
        BlazeLayout.render("layout", {content: "panels"});
    }
});

FlowRouter.route('/icons', {
    name: 'Icons',
    action: function() {
        BlazeLayout.render("layout", {content: "icons"});
    }
});

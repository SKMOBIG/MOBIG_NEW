First run

File system or server

To run HTML/jQuery version of LUNA you can run index.html file in your browser or run a local server under Luna_Full_Version_HTML folder.

Layout structure

Luna layout

Main layout are created with few main elements:

  1. wrapper main page wrapper
  2. navbar top header with menu
  3. left menu left sidebar navigation.
  4. content main container for page elements.
    <body>

    <!-- Wrapper-->
    <div class="wrapper">

        <!-- Header-->
        <nav class="navbar navbar-default navbar-fixed-top">
            ...
        </nav>

        <!-- Navigation-->
        <aside class="navigation">
            ...
        </aside>

        <!-- Main content-->
        <section class="content">
            ...
        </section>

    </div>
    <!-- End wrapper-->

    </body>

Luna styles

Third party styles

head contains list for all CSS files of theme and for all plugins (vendor - used in view when specific plugin need it). In your app you can choose only those files that you will use. 

<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <link href='http://fonts.googleapis.com/css?family=Roboto:300,400,500,700,900' rel='stylesheet' type='text/css'>

    <!-- Page title -->
    <title>LUNA | Responsive Admin Theme</title>

    <!-- Vendor styles -->
    <link rel="stylesheet" href="vendor/fontawesome/css/font-awesome.css"/>
    <link rel="stylesheet" href="vendor/animate.css/animate.css"/>
    <link rel="stylesheet" href="vendor/bootstrap/css/bootstrap.css"/>

    <!-- App styles -->
    <link rel="stylesheet" href="styles/pe-icons/pe-icon-7-stroke.css"/>
    <link rel="stylesheet" href="styles/pe-icons/helper.css"/>
    <link rel="stylesheet" href="styles/stroke-icons/style.css"/>
    <link rel="stylesheet" href="styles/style.css">
</head>
Font Awesome, Animate, Bootstrap, pe-icons, stroke-icons and style.css minimum required files for Luna !

Luna scripts

Third party scripts

At the bottom of each file there are located third party scripts as well as Luna script: 

<!-- Vendor scripts -->
<script src="vendor/pacejs/pace.min.js"></script>
<script src="vendor/jquery/dist/jquery.min.js"></script>
<script src="vendor/bootstrap/js/bootstrap.min.js"></script>

<!-- App scripts -->
<script src="scripts/luna.js"></script>

Luna local scripts

Initialise third party plugins

In some of the files where we use third party plugin initailise script is located at the bottom of the file in script section. For example: 

<script>
    $(document).ready(function () {

        // Activate Nestable for list 1
        $('#nestable').nestable({
            group: 1
        });

    });
</script>
Once you add link to the third aprty library you can initialise and put your code for plugins in local script section.
